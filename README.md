A lightweight, browser-based tool designed to remove solid colored backgrounds (Green Screen) from images instantly. Built with pure JavaScript and Tailwind CSS, focused on performance and ease of use.

---

## Features

* **Auto & Manual Modes:**
    * **Magic Mode:** Click once to automatically detect and remove the background color.
    * **Global Mode:** Select a specific color to remove it from the entire image.
* **Fine-Tuning Controls:** Adjust color tolerance, erode edges, and feather (blur) borders for a perfect cutout.
* **Zoom & Pan:** Easily navigate through high-resolution images.
* **Undo/Redo:** Made a mistake? Step back or forward with ease (supports shortcuts `Ctrl+Z` / `Ctrl+Y`).
* **Mobile Friendly:** Works smoothly on desktop and mobile devices.
* **Privacy First:** All processing happens in your browser. No images are uploaded to any server.

## How to Use

You don't need to install anything! This is a static web application.

1.  **Clone or Download** this repository.
2.  Open the folder.
3.  Double-click `index.html` to open it in your web browser.
4.  Upload an image and start editing!

## Tech Stack

* **HTML5 Canvas** (for image processing)
* **Tailwind CSS** (for styling via CDN)
* **Vanilla JavaScript** (no heavy frameworks)
